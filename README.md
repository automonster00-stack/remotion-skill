# 🎬 Remotion Skill

יצרתי לך skill מקיף לעבודה עם Remotion - פריימוורק ליצירת סרטונים עם React!

## 📦 הקובץ

**remotion.skill** - הקובץ הארוז מוכן להתקנה

## 🎯 מה ה-Skill הזה מאפשר?

- ✨ יצירת פרויקטים חדשים עם templates שונים
- 🎥 רינדור סרטונים מקוד React
- 🎨 אנימציות מתקדמות עם interpolation ו-spring
- 🎵 סנכרון עם אודיו
- 📊 סרטונים מונעי-data
- 🤖 אינטגרציה עם Claude Code
- 3️⃣ תמיכה ב-Three.js ל-3D

## 📚 מה כלול בפנים?

### SKILL.md - המדריך הראשי
- התקנה מהירה
- יצירת פרויקט חדש
- רינדור סרטונים
- טכניקות אנימציה
- עבודה עם אודיו ונכסים
- סרטונים מונעי-data
- פתרון בעיות נפוצות

### references/advanced-examples.md - דוגמאות מתקדמות
- אנימציות מורכבות עם spring
- layouts דינמיים
- סנכרון אודיו ווויזואליזציות
- עבודה עם Three.js
- compositions דינמיים
- אופטימיזציות ביצועים
- רינדור בשרת (AWS Lambda)

### scripts/create_and_render.sh - כלי עזר
סקריפט bash נוח לביצוע משימות נפוצות:
```bash
# יצירת פרויקט חדש
./create_and_render.sh create my-video hello-world

# רינדור סרטון
./create_and_render.sh render src/index.ts MyVideo output.mp4

# רינדור תמונה בודדת
./create_and_render.sh still src/index.ts MyVideo frame.png --frame=30

# הפעלת preview
./create_and_render.sh preview ./my-video
```
